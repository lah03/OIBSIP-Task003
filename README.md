
## Project Overview

Car price prediction is an important application of Machine Learning. The price of a car depends on many factors such as the car brand, year of manufacture, fuel type, transmission type, kilometers driven, and current market price.

This project builds a Machine Learning model to predict the selling price of used cars using a dataset from Kaggle.

---

## Dataset

The dataset used in this project is **Car Price Prediction (Used Cars)** dataset downloaded from Kaggle.

### Dataset Features

* Car Name – Name of the car
* Year – Year of manufacture
* Selling Price – Price at which the car is being sold
* Present Price – Current ex-showroom price
* Driven kms – Total kilometers driven
* Fuel Type – Petrol / Diesel / CNG
* Selling Type – Dealer or Individual
* Transmission – Manual or Automatic
* Owner – Number of previous owners
